# Простой банк для изучения Go backend разработки

Данное приложение представляет собой простой банк, разработанный для изучения backend-разработки на Go.

## Основные функции

1. **Создание и управление аккаунтами:** регистрация пользователей с указанием имени владельца, баланса, валюты и других параметров.
2. **Учёт изменений баланса:** фиксация всех операций, влияющих на баланс счёта.
3. **Денежные переводы между счетами:** выполнение транзакций между аккаунтами.
